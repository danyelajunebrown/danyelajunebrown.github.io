# Tech Context

## Technologies Used

### Frontend
- **HTML5** - Single page application structure
- **CSS3** - Styling with flexbox, grid, animations
- **Vanilla JavaScript** - All application logic

### External Libraries
- **Supabase JS SDK** (v2.39.7) - Authentication & database
- **html2canvas** (v1.4.1) - DOM to canvas capture for PDF
- **jsPDF** (v2.5.1) - PDF generation

### Development Environment
- Visual Studio Code
- Git for version control
- Browser-based testing (Chrome/Safari)

## Setup
- Single `index.html` file with embedded CSS and JS
- CDN-loaded dependencies (no npm build required)
- Supabase project for backend

## Browser APIs Used
- Canvas API (via html2canvas)
- Drag and Drop API
- Touch events for wheel interaction
- LocalStorage (via Supabase)

## Constraints
- Must work in modern browsers
- Mobile-responsive design needed
- PDF must work across devices
